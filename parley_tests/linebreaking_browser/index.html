Parley ↔ Chrome line-breaking harness

  • Recorder — records every seed's Chrome break decision to a CSV corpus (copy buttons per font).
  • Fuzzer — sweeps seeds, comparing Parley against Chrome live and persisting only the failing seeds.
  • Inspect — renders one seed's Chrome layout and overlays where Parley breaks.

Browser line-breaking data collection

Browser line-breaking fuzzer

Sweeps seeds comparing Parley against Chrome live, persisting only failing seeds. Query params: ?start=S, ?count=N, ?reset=1.

Inspect a seed

Renders each font's case as Chrome lays it out (the visible wrap is Chrome's break) and overlays a red line where Parley breaks. Pass ?seed=N, and optionally ?width=S to force a width (in subpixels) instead of the tightened one.
